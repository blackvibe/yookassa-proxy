# YooKassa Proxy

Простой прокси-сервер для перенаправления запросов к API ЮКассы через российский хостинг.

## Локальный запуск

```bash
npm install
npm start
```

Сервер запустится на порту 3000 (или на порту из переменной окружения `PORT`).

## Деплой на Timeweb App Platform

1. Залейте этот репозиторий на GitHub
2. В Timeweb App Platform создайте новое приложение
3. Выберите этот репозиторий
4. Настройки:
   - **Build Command**: `npm install` (или оставьте пустым)
   - **Start Command**: `npm start`
5. Деплойте

## Использование

После деплоя Timeweb выдаст домен (например, `frontend-app-88.twc1.net`).

Меняйте в своем коде:

**Было:**
```
https://api.yookassa.ru/v3/payments
```

**Стало:**
```
https://frontend-app-88.twc1.net/v3/payments
```

Все заголовки авторизации и тело запроса остаются без изменений.

## Требования

- Node.js тариф на Timeweb (Frontend с поддержкой SSR)
- ~20 МБ дискового пространства
- Достаточно для ~50k запросов/месяц
